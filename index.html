JS & jQuery basics

Operators

An operator is a special character (or characters) that indicates an action to be performed (+,-,*,/,%).

Variables

A variable is just a container that stores some information such as a number, an object, or a string.

String

A string is a set of characters enclosed in quotations.

Boolean

A boolean can be one of two values: true or false.

Arguments

A parameter is essentially just an empty variable that we can use inside the function while the argument is the value we pass to that variable.
